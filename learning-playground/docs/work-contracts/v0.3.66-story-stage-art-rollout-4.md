# v0.3.66 Story Stage Production-Art Rollout Slice 4

One bounded post-approval rollout under
`docs/contracts/art-production-assets.contract.md`: complete the production-art
journey for Princess Poppy's Special Delivery story in Cloud Village. This is
not a full Story Stage art migration.

## Before Code

### Root Cause

The merged production-art maps cover Princess Poppy's Special Delivery journey
only in the Enchanted Forest. The current story contract also supports that
same Poppy + Special Delivery selection in Cloud Village, but all eight scene
keys there still render geometry assembled inside
`src/modules/story-stage/story-art.ts`. The child's selected Cloud Village
setting therefore does not yet produce a separately editable, locally exported
journey with asset-level provenance and responsive visual evidence.

The cause is not story compatibility, selection, resolution, narration,
branching, history, speech, persistence, or layout. The runtime has no closed
production-asset map for the exact Poppy + Cloud Village + Special Delivery
context, and the corresponding eight editable sources and exports do not
exist.

### Correct Fix Must Touch

  `delivery-meadow`, `delivery-protect`, `delivery-ending-high`, and
