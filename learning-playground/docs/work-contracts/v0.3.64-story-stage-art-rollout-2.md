# v0.3.64 Story Stage Production-Art Rollout Slice 2

One bounded post-approval rollout under
`docs/contracts/art-production-assets.contract.md`: complete the production-art
journey for Princess Poppy's Special Delivery story in the Enchanted Forest.
This is not a full Story Stage art migration.

## Before Code

### Root Cause

The owner-approved Princess Poppy and Enchanted Forest production-art system now
covers the complete Lost Friend family, but the only other story family that the
runtime permits for that exact child-selected character and setting is still
drawn by geometry assembled inside `src/modules/story-stage/story-art.ts`.
Special Delivery therefore changes back to the older in-code visual system even
though the child kept the same Poppy and forest choices.

The cause is not story resolution, branch logic, narration, layout, or missing
content. The production asset lookup is intentionally a closed map containing
only Lost Friend art keys. None of Special Delivery's eight unique art keys has
a separately editable Inkscape source, local production export, runtime mapping,
or cohesive provenance entry.

### Correct Fix Must Touch

- `design-source/story-stage/special-delivery-poppy-forest/` (new): eight
  layered, editable Inkscape SVG sources for `delivery-intro`,
  `delivery-problem`, `delivery-route`, `delivery-bridge`, `delivery-meadow`,
  `delivery-protect`, `delivery-ending-high`, and `delivery-ending-wagon`.
- `public/assets/images/` (new exports): eight optimized, local 400x250 SVGs
  with stable `story-stage-<art-key>-poppy-forest.svg` names matching those
  scene beats.
- `src/modules/story-stage/story-art.ts`: add a closed Special Delivery local
  asset map and consult it only when `characterArt === "poppy"`,
  `settingArt === "forest"`, and the art key exists in that map. Preserve the
  complete approved Lost Friend map and every unmatched dynamic builder and
  fallback.

  keys resolve to their intended local decorative assets; walk both authored

  every source/export/review path, Category A rights, approved-direction

- Preserve the approved Poppy/forest system's original purple ink, warm flat
  fills, rounded silhouettes, layered forest depth, pale path, gentle light,
  large readable characters, and foreground framing.
- Each scene must communicate its authored beat before narration: special-day
  readiness, entrusted berry delivery, bridge-or-meadow route choice, careful
  bridge crossing, quiet butterfly meadow, high-or-wagon entrance choice, exact
  carry-high celebration, and exact wagon celebration.

  forest palette, sunshine-berry basket, scale language, and left-to-right

  the route, protection choice, and exact payoff.

  may crop, but Poppy, basket, bridge, route fork, butterflies, wagon, party
  friends, and the chosen ending action must remain legible in the existing
  400x250 stage.

  remain the accessible source of instructions.
