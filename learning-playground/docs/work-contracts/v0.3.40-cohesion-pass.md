surfaces still belong to the old dark system: the home grid floats **dark

Cafe completion still bursts **emoji glyphs** (🎉 ✨ ⭐ …), and the

## Contract Amendments

- **Domain gate corrected `shapes` → `spatial`.** The contract named the
  gate `domain === 'shapes'`, but the three Shapes activities declare
  `domain: "spatial"` (the only spatial-domain content) — the typechecker
  rejected the non-overlapping comparison, which would otherwise have been
  a silent no-op tint. Gate and its source-assertion test use `'spatial'`.
- **Home cards made opaque after live capture review.** The warm accent
  gradients ended in translucent rgba, so the cafe backdrop bled through
  the controls (window frame visible through the Words card). A solid
  cream base layer under each gradient restores control-over-scene
  hierarchy. Caught by looking, not by tests.
- **Greeting given a cream halo.** Making "Let's play!" legible ink exposed
  a mild overlap with the backdrop's Bear Cafe sign on wide viewports; a
  soft cream text-shadow separates it. Same capture review.
